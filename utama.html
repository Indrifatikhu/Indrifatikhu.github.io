Description

Liscio Stationery is a personal shop that sells several kinds of stationery. This shop was established on August 1, 2020 because shop owners really like stationery. The shop owner was also experiencing a monetary crisis at that time, so he asked his mother for money and started a business.

How To Order

  • 1. Open the Shopee application
  • 2. Find the store name using the search engine at the top of Shopee's first page
  • 3. If you find it, click on the store to go to the store page
  • 4. Select the item which you want to buy
  • 5. Go to basket
  • 6. Checkout the items
